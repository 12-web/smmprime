  • Установить приложение для IOS

    • 1. В Safari откройте страницу smmprime.com
    • 2. Нажмите кнопку "Поделиться"
    • 3. Выберите пункт На экран "Домой"
    • 4. Введите название для приложение и нажмите Добавить
    • 5. Теперь у вас на рабочем столе установлено web-приложение Smmprime!

  • Установить приложение для Аndroid

    • 1. В Chrome откройте страницу smmprime.com
    • 2. Нажмите кнопку настроек (три точки)
    • 3. Выберите пункт Добавить на гл. экран
    • 4. Введите название для приложение и нажмите Добавить
    • 5. Теперь у вас на рабочем столе установлено web-приложение Smmprime!
